# Rescue Bird Product Idea

## Vision

Rescue Bird is a mobile-centric emergency response platform for Bangladesh, focused on fast citizen alerting, team dispatch coordination, and transparent operational oversight.

## Target Context

- Primary usage through mobile webview
- High urgency interactions with minimal input friction
- Dhaka-first operational model with expandable area logic

## Account Roles

- `admin`: governance, audit visibility, platform operations
- `rescue_team`: service coverage setup, incident response coordination
- `team_staff`: field-level response and communication
- `user`: emergency requester and communication initiator

## Core Journey

1. User submits emergency alert with area, coordinates, short note, and optional voice note URL.
2. Platform matches the best rescue team using coverage areas + base location radius logic.
3. Team/staff accepts and resolves incidents while communicating with user in the team thread.
4. Admin can audit user/message activity and identify who sent what.

## UX Principles Used In Redesign

- Mobile-first hierarchy with role-focused tabs
- Fast-path emergency actions with clear visual priority
- Low cognitive load form layouts and clean status signaling
- Consistent card system, readable spacing, and responsive components
- Operational transparency for team and admin workflows
